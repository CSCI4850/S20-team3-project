# S20-team3-project

Required Packages:

    Python    == 3.7
    gym-retro >= 0.7.0
    
And all their dependencies

Create a 3.7 virtualenv

    virtualenv --python=/path/to/python3.7 venv # /usr/bin/python3.7 for example

import the rom:

    python3 -m retro.import 'Galaga - Demons of Death (USA).nes'

to run:
	source venv/bin/activate
	./basic.py
